# Kittygram - сервис для публикации фотографий котов

## Описание проекта
Kittygram - это веб-приложение для публикации и просмотра фотографий котов. 
Пользователи могут загружать изображения, ставить лайки и комментировать.

## Функционал
- Регистрация и авторизация пользователей
- Загрузка фотографий котов
- Просмотр ленты с фотографиями
- Лайки и комментарии

## Технологический стек
- Backend: Python 3.9, Django 3.2, Django REST Framework
- Frontend: React, Node.js 18
- База данных: PostgreSQL 13
- Веб-сервер: Nginx
- Контейнеризация: Docker, Docker Compose
- CI/CD: GitHub Actions
